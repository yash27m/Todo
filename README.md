A simple and clean Todo application built using Flutter.  
This app helps users manage daily tasks efficiently with local data storage.

## Features
- User Login & Registration
- Add, Edit, and Delete Tasks
- Local storage using Hive
- Clean and modern UI
- Data persists after app restart

## Tech Stack
- Flutter
- Dart
- Hive (Local Database)

## How to Run
1. Clone the repository  
   git clone https://github.com/yash27m/Todo.git  

2. Open the project folder  
   cd flutter_todo_app

3. Install dependencies  
   flutter pub get

4. Run the app  
   flutter run
